## How to Use

1. **Copy & paste** any prompt directly into GPT-4 or Claude as the **system prompt**.
2. **Replace placeholders** in `[SQUARE_BRACKETS]` with your actual values.
3. **Chain prompts** — e.g. run `Markdown Outline Builder` first, then `Full Blog Post + JSON Package`.
4. **Import the JSON** into your CMS or prompt management tool for team-wide use.

## Guardrail Conventions

- `[VERIFY]` — fact needs external confirmation before publishing
- `[PLACEHOLDER]` — dynamic input required
- `[QUOTE TO BE VERIFIED]` — confirm with original source
- `[PRICE — CHECK SITE]` — never assume pricing; link to official page
- `[INTERNAL LINK: ...]` — place a real internal link here
